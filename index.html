Procesamiento de Ventas con PySpark & Streamlit

Un pipeline de datos eficiente para cargar archivos CSV diariamente, realizar cálculos distribuidos, almacenar en formato estructurado Parquet y visualizar los resultados en un panel interactivo premium.

Guía de Implementación Paso a Paso

📂 1. Carga del Archivo CSV de Ventas

Iniciamos una SparkSession y cargamos el archivo CSV de ventas diarias. Utilizamos la inferencia automática del esquema de datos para asignar los tipos de datos correspondientes de manera dinámica.

Python (pyspark)
from pyspark.sql import SparkSession

# Iniciar la sesión de Spark
spark = SparkSession.builder \
    .appName("AnalisisVentas") \
    .master("local[*]") \
    .getOrCreate()

# Cargar CSV con detección de cabecera y esquema automático
df = spark.read \
    .option("header", "true") \
    .option("inferSchema", "true") \
    .csv("ventas.csv")

🧮 2. Cálculo de la Columna total_venta

Calculamos el importe total de cada venta multiplicando la columna cantidad por precio empleando operaciones columnares nativas de Spark, optimizadas para procesamiento paralelo.

Python (pyspark)
from pyspark.sql.functions import col

# Agregar columna total_venta (cantidad * precio)
df_procesado = df.withColumn(
    "total_venta", 
    col("cantidad") * col("precio")
)

# Mostrar resultados procesados en consola
df_procesado.show()

💾 3. Persistencia de Datos en Parquet

Guardamos el DataFrame resultante en formato de columnas comprimido **Parquet**. Esto reduce drásticamente el espacio de almacenamiento y acelera las futuras consultas analíticas de lectura.

Python (pyspark)
# Guardar en formato columnar Parquet sobrescribiendo archivos previos
output_path = "output/ventas_procesadas.parquet"

df_procesado.write \
    .mode("overwrite") \
    .parquet(output_path)

📊 4. Dashboard Interactivo de Visualización

Creamos un dashboard elegante con **Streamlit** y **Plotly** para importar el conjunto Parquet procesado, exhibir métricas clave (KPIs) y gráficos interactivos con soporte multidispositivo.

Python (streamlit)
import streamlit as st
import pandas as pd
import plotly.express as px
import glob

# Cargar archivos parquet procesados
archivos = glob.glob("output/ventas_procesadas.parquet/*.parquet")
df = pd.concat([pd.read_parquet(f) for f in archivos])

# Layout e interfaz de Streamlit
st.title("📊 Dashboard de Ventas")
st.metric("Ingresos Totales", f"${df['total_venta'].sum():,}")

fig = px.bar(df, x="producto", y="total_venta", color="producto")
st.plotly_chart(fig)

Beneficios del Stack Tecnológico

Procesamiento Distribuido

Spark ejecuta transformaciones perezosas (lazy transformations) mapeando tareas concurrentes a nivel de CPU local o clusters distribuidos.

🗂️

Almacenamiento Columnar

Parquet almacena esquemas integrados comprimidos y soporta proyecciones y pushdown-predicates para leer exclusivamente los datos necesarios.

🎨

Experiencia de Usuario Premium

Streamlit proporciona renderizado inmediato y reactivo acoplándose sin fisuras a motores gráficos de alta gama como Plotly.
